Hello World
    เมื่อเราสร้างโปรเจค rust ใหม่ เรามักจะเห็นชุดคำสั่งเหล่านี้เป็นอันดับแรก
        fn main() {
            println!("Hello, World");
        }

    fn หมายถึง functions
    main เป็นฟังก์ชันแรกที่จะถูกเรียกใช้งานจากคอทไพลเลอร์
    {} เรียกว่า code block จะเป็นส่วนที่เราจะเขียนชุดคำสั่งเอาไว้

    {} ข้างใน println!() ใช้สำหรับบอก compiler ว่าจะมีการวางข้อมูลแทรกตรงที่ระบุ

    -> เรียกว่า skinny arrow สำหรับระบุว่่าฟังก์ชันนั้นจะส่งคืนเป็นชนิดข้อมูลอะไร
    โดยถ้าหากข้อมูลของในไม่มี ; ปิดท้าย จะเป็นบอกคอมไพลเลอร์ว่าจะมีการส่งคืนข้้อมูลตัวนี้ออกไป
    โดยเราสามารถมี return ได้ 2 แบบ คือ
        8
        return 8;

    การประกาศตัวแปร
        เราจะใช้ let ในการประกาศตัวแปรขึ้นมา (สิ่งที่จะเอาไว้เก็บข้อมูลในโปรแกรมของเรา)
        โดยตัวแปร จะถูกสร้างขึ้นและถูกทำลาย ภายใน code block นั้น ๆ
            fn main() {
                {
                    let num = 10;
                }
                println!("{}", num);
            }
        จากตัวอย่างชุดคำสั่งดังกล่าว จะไม่สามารถใช้งาน num ภายใน println!() ได้ 
        เนื่องจาก num ถูกทำงายไปเมื่อ code block หมดแล้ว
