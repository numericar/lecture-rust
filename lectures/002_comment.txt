Comment
    ใช้สำหรับเขียนข้อความอธิบายอะไรบางอย่าง
    โดยระบบจะไม่อ่านบรรทัดนั้นในขั้นตอนการ compile code

    โดยมักใช้เพื่ออธิบายถึงชุดคำสั่งที่กำลังทำงาน
    (หลาย ๆ ครั้งเรามักจะมีชุดคำสั่งที่สุดยอด แต่ลืมไปว่าเขียนมันไปทำไมนะ ?)

    โดยการ comment จะใช้สัญลักษณ์ //
    
    และเราสามารถ comment หลายบรรทัดด้วยโดยใช้สัญลักษณ์ /* */
